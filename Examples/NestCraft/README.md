# NestCraft

### Game on NestEngine 

This mini game is written on the NestEngine

![Alt text](Screenshot.png "Screenshot")

### Windows
Build with CMake:
``` sh
git clone https://github.com/bogdanmikh/Nest.git
mkdir build && cd build
cmake -G "Visual Studio 16 2019" ..
start NestCraft.sln
```

### Linux or Macos
Build with CMake:
``` sh
git clone https://github.com/bogdanmikh/Nest.git
mkdir build && cd build
cmake ..
make NestCraft
```
