# NestPBR
## Game on NestEngine 

This game implements lighting

![Alt text](Screenshot.png "Screenshot")

### Windows
Build with CMake:
``` sh
git clone https://github.com/bogdanmikh/Nest.git
mkdir build && cd build
cmake -G "Visual Studio 16 2019" ..
start NestPBR.sln
```

### Linux or Macos
Build with CMake:
``` sh
git clone https://github.com/bogdanmikh/Nest.git
mkdir build && cd build
cmake ..
make NestPBR
```
